Trend Review · 전체 페이지 디자인 시안

AEditorial · 학술지풍

크림 배경 + 명조 제목 + 넉넉한 여백. 차분·권위·클래식.
BModern Gradient · 모던 대시보드

그라데이션 헤더 + 유리효과 통계 + 둥근카드·그림자. 화사·앱 느낌.
CClinical Pro · 다크+틸 전문가풍

다크 헤더 + 틸 강조 + 모노스페이스 + 좌측 컬러바. 데이터밀도·프로.
DBento · 컬러 타일

둥근 컬러 타일(보라·초록·노랑) + 큰 번호. 발랄·시원시원·트렌디.
ENewsprint · 신문/브루탈리즘

흑백 + 두꺼운 검은 테두리 + 큰 대문자 헤드라인. 강렬·인쇄물 느낌.
FSoft Calm · 부드러운 파스텔

따뜻한 베이지 + 파스텔 태그 + 라운드. 편안·가독성·노션 느낌.
GDark Neon · 다크+네온

검은 배경 + 라임 네온 강조 + 글로우. 강한 모던·테크 느낌.
