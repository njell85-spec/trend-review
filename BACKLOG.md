# Backlog — 나중에 한번에 작업

## 디자인 개선 (PeterJ 요청 2026-06-29, "나중에 한번에")
1. **발행 연-월까지 표기**: 현재 저널·연도만(예: `N Engl J Med · 2026`) → 가능하면 **연-월**(예: `N Engl J Med · 2026.03`).
   - 데이터는 이미 있음(`DataCollectorAgent._parsePubDate` = YYYY-MM-DD). 표시부(`GitHubPublisher`)와 오늘 시드만 반영하면 됨.
2. **제목 한글/영문 동일 폰트 크기**: 현재 한글(`.ttl` 18px bold) 크고, 영문 원문(`.ttle` 11.5px) 작음 → **둘 다 같은 크기**로.
3. **누적 리스트 표 + 읽음 체크박스**: 페이지 하단에 전체 논문 누적 테이블(날짜·저널·제목·PMID), **우측에 읽음/안읽음 체크박스**.
   - 정적 사이트라 체크 상태는 **localStorage(PMID 키)** 로 브라우저에 저장(서버 불필요).

> 위 3개는 데일리 자동화 토큰 세팅이 끝난 뒤 `GitHubPublisher` 한 번에 수정.
