⚠️ 알아둘 점

NotebookLM 자동 등록은 수동 모드(B5). 자동 등록이 인증 문제로 실패했지만 매월 1일 카톡 리마인더로 폰에서 소스 2개만 추가하면 됩니다(2클릭). 데이터 누락 없음.
구 토큰 폐기 권고 (보안). 예전 Trend_Review(대문자) 저장소용 옛 설정에 GitHub 토큰이 평문으로 남아 있음 → Developer settings → PAT (classic)에서 Revoke 권장.
자료화 버튼 본격 사용은 R5 이후. 실제 영상 생성+비공개 업로드(비용)라 품질 개선 전엔 1~2건만. 삭제 버튼은 자유.
